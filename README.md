# BibliotecaMultimediale

ENG
________________
A simple OOP exercise in Java, consisting in building the models for a Multimedia Library and being able to randomly generate objects instances out of them.

It uses java-faker library to generate some fake data.

IT
_____________________

Un semplice esercizio di OOP in Java, consiste nel realizzare dei modelli per la gestione di una biblioteca di file multimediali e poter generare istanze di questi oggetti in modo random.

Utilizza la libreria java-faker per generare dei dati fasulli.
